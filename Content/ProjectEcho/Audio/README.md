# Audio

## Purpose

Project Echo audio content: ambient beds, music, SFX, and voice.

## Allowed Assets

- Sound Waves, Sound Cues, MetaSounds (as adopted), audio data assets
- Organized under Ambient / Music / SFX / Voice

## Examples

- `Ambient/`, `Music/`, `SFX/`, `Voice/`

## Best Practices

- Prefer project buses / consistent naming before cue sprawl grows.
- Keep dialogue and VO under `Voice/`.
- Large audio files should follow Git LFS policy (`*.wav` tracked).
- Do not implement full audio directors here until an Audio system mission is authorized.
