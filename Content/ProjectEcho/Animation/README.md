# Animation

## Purpose

Animation assets for characters and props: sequences, montages, blend spaces, Control Rig assets as needed.

## Allowed Assets

- Animation Sequences, Montages, Blend Spaces, Anim Blueprints (`ABP_` when used), Control Rig assets

## Examples

- Player locomotion set, Witness animation set (when authorized)

## Best Practices

- Keep Anim Blueprints focused; avoid gameplay decision-making that belongs in Actor Components.
- Name consistently and group by character.
- Do not import large animation sets without LFS-aware commits.
