# Developer

## Purpose

Local/experimental Project Echo developer sandbox inside the project content root.

## Allowed Assets

- Temporary test assets clearly marked as experimental

## Examples

- Throwaway meshes or test maps that are not production-ready

## Best Practices

- Do not ship production systems from this folder.
- Prefer `Levels/Test/` for formal test maps.
- Per-user Epic `Content/Developers/` is gitignored; this folder is project-scoped — use sparingly.
