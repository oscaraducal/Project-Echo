# Levels

## Purpose

All Project Echo maps, organized by production stage.

## Allowed Assets

- `LV_` levels / umaps
- Level instances / streaming setup assets that belong to these maps

## Examples

- `Prototype/LV_Prototype`
- Future: Persistent, Test, VerticalSlice maps

## Best Practices

- Use the stage folder that matches intent (`Prototype`, `Test`, `VerticalSlice`, `Persistent`).
- Keep prototype maps lean; promote content intentionally toward Vertical Slice.
- Do not delete or rename maps without TD approval.
- Prefer Project Echo maps as Editor/Game defaults (already configured).
