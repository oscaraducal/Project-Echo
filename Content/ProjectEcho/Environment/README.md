# Environment

## Purpose

World art and environment content for the Facility and related spaces.

## Allowed Assets

- Static meshes (`SM_`), materials (`M_` / `MI_`), textures (`T_`)
- Architecture, props, decals, foliage, environment materials

## Examples

- `Architecture/`, `Props/`, `Meshes/`, `Materials/`, `Textures/`, `Decals/`, `Foliage/`

## Best Practices

- Keep gameplay logic out of pure environment meshes; use Gameplay Blueprints for interactivity.
- Follow Asset Master List / replacement plan for placeholder art.
- Prefer modular kits and consistent scale.
