# Cinematics

## Purpose

Sequencer / cinematic content for scripted moments, cameras, and narrative presentation.

## Allowed Assets

- Level Sequences, cinematic cameras, related cinematic assets

## Examples

- Future intro/outro or Witness encounter sequences (when authorized)

## Best Practices

- Prefer soft references and streamed levels where appropriate.
- Keep gameplay-critical logic out of sequences when a system should own it.
- Coordinate with Narrative / Level Design docs before building long cinematics.
