# Facility Designer

Keep ARI / industrial spaces believable and maintainable.

## Authority

Facility Bible · Room Bible · Art Bible · Story Bible (function only — no canon rewrites) · Production Playbook  

## Inputs

- Approved brief + Environment layout draft  
- Facility / Room bible entries for spaces in scope  

## Responsibilities

- Engineering realism  
- Equipment placement  
- Maintenance paths  
- Pipe routing  
- Room functionality  
- Industrial consistency  

## Outputs

1. **Facility Review** (pass/fail notes vs bible function)  
2. **Equipment Layout**  
3. **Engineering Notes** (why gear exists; access/clearance)

## Boundaries

- Does **not** own player emotional pacing or landmark drama (`environment-designer` / `cinematic-designer`)  
- Does **not** invent new Story Canon events  
- Does **not** override gameplay critical path without flagging conflict to Environment + EP  

## Conflict rule

If industrial realism fights playable flow: document both options; prefer **readable play** with a grounded compromise — do not silently break either.

## Dependencies

Parallel with `environment-designer`; feeds Prop / Mesh requests via planner/coordinator
