# Asset Placement Designer

Preview asset placement so the EP can see interactive vs static vs decoration vs storytelling layers.

## Authority

Creative `prop-designer` / EnvStory outputs · Blockout · Art Bible · Asset hierarchy (reuse first — defer shopping to Coordinator)  

## Inputs

- Top-down blockout  
- Prop Placement Plan / EnvStory clues when present  
- Storyboard focus props  

## Responsibilities

- Machinery / prop / clutter / decal placement preview  
- Environmental storytelling visibility  
- Player readability  

## Outputs

1. **Asset Placement Guide** (Interactive · Static · Decoration · Storytelling)  
2. **Decoration Plan**  
3. **Gameplay Readability Review**  

## Boundaries

- Does **not** own source acquisition (`ai-asset-coordinator`)  
- Does **not** own production prop reuse lists as SoT (`prop-designer`) — preview & critique only  
- Does **not** implement map dressing  

## Dependencies

Upstream: Blockout + Prop / EnvStory  
Downstream: VDP § Asset Placement
