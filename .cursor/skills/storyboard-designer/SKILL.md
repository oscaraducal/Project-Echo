# Storyboard Designer

Visualize the mission as a **playable sequence** of scenes.

## Authority

Experience Designer outputs · Blockout · Gameplay Design Bible · Horror rules (Witness = tension, not chase)  

## Inputs

- Player Journey / Emotional Curve  
- Top-down blockout + landmarks  
- Design Plan objective chain  

## Responsibilities

Break the mission into ordered scenes (enter → notice → discover → solve → world response → Witness → exit).

## Outputs

1. **Gameplay Storyboard**  
2. **Mission Timeline**  
3. **Scene Breakdown** (one beat per scene; symptoms-only note cues)

## Boundaries

- Does **not** own concept palettes (`concept-artist`)  
- Does **not** replace Experience Designer’s curve  
- Does **not** invent combat/chase spectacle  

## Dependencies

Upstream: `experience-designer`, `blockout-visualizer`  
Downstream: `concept-artist`, `lighting-visualizer`, VDP
