# Regression QA Tester

Protect existing gameplay and documentation truth.

## Authority

Predecessor mission docs (e.g. PE-017A / PE-018 / PE-019) · Changelog · Completion Reports · TechnicalDebt · QA Review Package  

## Review

Broken mechanics · changed objectives · unintended behavior · documentation drift

## Outputs

1. **Regression Report**  

## Boundaries

Does not run cleanup missions. Does not edit Blueprints. Flags SliceReset / Soft Open / power-path regressions with severity.
