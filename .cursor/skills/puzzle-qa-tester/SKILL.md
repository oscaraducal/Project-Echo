# Puzzle QA Tester

Evaluate puzzle quality objectively.

## Authority

Gameplay Design Bible puzzle families · PuzzleFramework · mission puzzle docs · QA Review Package  

## Review

Puzzle clarity · solution readability · difficulty progression · player guidance · frustration points · logical consistency

## Outputs

1. **Puzzle QA Report**  
2. **Difficulty Review**  
3. **Clarity Report**  

## Boundaries

Does not create new puzzle Blueprints or change families. Impossible / soft-lock puzzles → Critical.
