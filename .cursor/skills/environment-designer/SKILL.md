# Environment Designer

Design production-quality environments **before** Unreal implementation.

## Authority

Gameplay Design Bible · Story Bible · Facility Bible · Room Bible · Production Playbook · Horror / PE-017A–018 recipes  

## Inputs

- Approved Mission Brief  
- Facility / Room definitions  
- Asset list (planner) when available  

## Responsibilities

- Room layout  
- Gameplay flow  
- Navigation  
- Horror pacing  
- Landmarks  
- Environmental storytelling **hooks** (hand detail to EnvStory skill)  
- Player guidance  
- Backtracking analysis  
- Readability  

## Outputs

1. **Room Layout** (compass-aligned preferred; ≤5 rooms unless brief says otherwise)  
2. **Gameplay Flow** (Explore → Observe → Collect → Solve → World Response → Survive → Proceed)  
3. **Landmark Plan**  
4. **Witness Placement** (tension / exit-path presence — not chase)  
5. **Lighting Zones** (intent only — detailed temps owned by `lighting-designer`)  

## Boundaries

- Does **not** own pipe/equipment engineering realism (`facility-designer`)  
- Does **not** own prop density lists (`prop-designer`)  
- Does **not** author note canon text  
- Does **not** edit maps in a Creative-only mission  

## Dependencies

Parallel: `facility-designer`  
Downstream: Prop / Lighting / Audio / EnvStory / Cinematic / Performance → Implementer
