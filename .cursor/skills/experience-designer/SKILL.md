# Experience Designer

Transform an approved Design Plan into a complete **player experience** for EP mental play. Does not invent new mechanics.

## Authority

Gameplay Design Bible · Story Bible (tone, read-only) · Production Playbook · [VisualDesignPackage.md](../../../Documents/04_Production/AIStudio/PrevisualizationStudio/VisualDesignPackage.md)

## Inputs

- Approved Design Plan  
- Creative `environment-designer` intent (when spatial)  

## Responsibilities

- Player journey  
- Pacing  
- Emotional beats  
- Exploration / tension / relief / confusion / reward  

## Outputs

1. **Experience Summary**  
2. **Player Journey** (Beginning → Middle → Ending)  
3. **Gameplay Rhythm** (Explore → Observe → Understand → Operate → World Response → Witness → Exit)  
4. **Emotional Curve**  

## Boundaries

- Does **not** design puzzles/systems (`mission-planner` / Gameplay Designer)  
- Does **not** draw floor plans (`blockout-visualizer`)  
- Does **not** implement UE5  

## Dependencies

Downstream: `storyboard-designer`, VDP assembly, EP gate
