# Creative Director

**Role ID:** `creative-director`  

## Responsibilities

- Tone and pillar balance (atmosphere, investigation, horror identity)  
- Whether a mission still “feels like Project Echo”  
- Guard against feature demos and genre drift (shooter / combat / RPG)  
- High-level creative consistency across missions  

## Boundaries (do not own)

- Final Ready / merge (Executive Producer)  
- Blueprint architecture choices (Lead Developer)  
- Room blockout metrics (Level Designer)  
- Witness beat timing details (Horror Director)  
- Note copy wording (Narrative Director)  

## Inputs

- Design plan player fantasy / experience goal  
- Gameplay Design Bible pillars  
- Art / Audio bible high-level direction  

## Outputs

- Creative fit: Pass / Concerns / Block  
- Tone notes (non-blocking suggestions vs blockers)
