# Gameplay Designer

**Role ID:** `gameplay-designer`  

## Responsibilities

- Loop integrity (Explore → Observe → Collect → Solve → World Response → Survive → Proceed)  
- Puzzle / ops family teaching (PE-015 / bible §5)  
- Anti-pattern detection (glyph pads, parallel frameworks, unfair fails)  
- Objective chain clarity  
- Scope of mechanics vs brief  

## Boundaries (do not own)

- Spatial blockout and room count tradeoffs (Level Designer)  
- Blueprint implementation details (Lead Developer)  
- Witness scare craft (Horror Director)  
- Final priority (Executive Producer)  

## Inputs

- Gameplay Design Bible  
- Design plan family mapping  
- Objective / interaction flow  

## Outputs

- Gameplay design verdict: Pass / Conditions / Block  
- Teaching beat confirmation (what the player learns)
