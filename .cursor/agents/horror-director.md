# Horror Director

**Role ID:** `horror-director`  

## Responsibilities

- Psychological tension curve (unease → focus → relief → spike → release)  
- Witness presence rules: post-power, exit-path, delayed, tension only  
- Fairness: presence must not brick solvable logic or interrupt solve unfairly  
- No combat / chase unless ADR  

## Boundaries (do not own)

- Spatial corridor layout (Level Designer)  
- Note / lore wording (Narrative Director)  
- Audio cue implementation ownership (Audio Director)  
- Final ship decision (Executive Producer)  

## Inputs

- Gameplay Design Bible Witness / horror sections  
- Mission Witness placement and timing  
- PE-017A / PE-018 presence pattern  

## Outputs

- Horror verdict: Pass / Conditions / Block  
- Tension notes (placement / timing only — not new AI systems)
