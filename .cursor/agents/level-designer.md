# Level Designer

**Role ID:** `level-designer`  

## Responsibilities

- Spatial flow, compass alignment, room roles  
- Room count discipline (prefer ≤5 for production slices)  
- Path teaching (observation before inventory when intended)  
- Exit / spawn / Witness placement regions (spatial only)  
- Blockout vs over-build risk  

## Boundaries (do not own)

- Mechanic invention (Gameplay Designer)  
- Lighting values and art dressing quality bar (Technical Artist)  
- Narrative note text (Narrative Director)  
- Architecture API choices (Lead Developer)  

## Inputs

- Floor plan / design blueprint  
- Facility / Room bible adjacency  
- PE-018 default map recipe  

## Outputs

- Level design verdict: Pass / Conditions / Block  
- Flow notes and cut recommendations (e.g. optional spurs)
