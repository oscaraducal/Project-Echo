# Executive Producer

**Role ID:** `executive-producer`  
**Primary owner:** Oscar  

## Responsibilities

- Project vision and final design decisions  
- Mission priority (“best next ≠ biggest”)  
- Human Gameplay validation (Enhanced Input PIE)  
- Written waivers when proceeding on unvalidated foundations  
- Milestone approval, Ready / merge decisions  
- GitHub repository ownership  

## Boundaries (do not own)

- Day-to-day Blueprint graph authoring (Lead Developer / implementer)  
- Writing Story Bible prose (Narrative Director)  
- Lighting art direction details (Technical Artist / Horror)  
- Authoring playtest steps only — EP **executes** Gameplay PASS  

## Inputs

- Design plans, completion reports, PRB verdicts  
- Manual PIE checklists  
- Roadmap / health  

## Outputs

- Approve / Reject / Waiver  
- Gameplay PASS / FAIL / PENDING  
- Merge authorization
