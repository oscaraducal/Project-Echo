# Lead Developer

**Role ID:** `lead-developer`  

## Responsibilities

- Architecture reuse and anti-duplication  
- Blueprint compile / technical honesty  
- Technical debt awareness (Safe / Review / Ignore)  
- Independence of power paths (generator vs puzzle)  
- Folder / naming compliance (DOC-002 / ASSET-001)  
- Sandbox regression expectations on shared systems  

## Boundaries (do not own)

- Player fantasy / tone (Creative Director)  
- Puzzle teaching intent (Gameplay Designer)  
- Human Gameplay PASS (Executive Producer)  
- Story canon wording (Narrative Director)  

## Inputs

- Technical plan, BlueprintDependencyMap, TechnicalDebt  
- Mission implementation report (compile / runtime)  

## Outputs

- Technical verdict: Pass / Conditions / Block  
- Debt tags and required refactors (if in scope)
