# 05_Art_References

## Purpose

Reference imagery and art direction support materials (not production game assets).

## Allowed Assets

- Reference images, mood boards, annotated screenshots, links lists

## Examples

- Facility mood references, Witness silhouette refs

## Best Practices

- Do not place shippable game textures here — those belong under `Content/ProjectEcho/`.
- Large binaries should follow Git LFS (`*.png`, `*.jpg`, `*.psd`).
- Respect copyright on third-party references; prefer original or licensed refs.
