## Technique takeaways applied to Medical Wing

### 1. Process spine, not lobby clone
Isolation-style circulation: player reads **procedure** (triage → filing → isolation → observation) instead of another hub-and-spokes checkpoint. Pinch at Isolation Anteroom = Alien-style gate, clinical language.

### 2. Care interrupted, not blood spectacle
Exam lamp + drawn curtain + empty bed reads mid-procedure evacuation (SOMA / Outlast hospital craft without gore identity). Sharps / seal breach / PPE cart = contamination story.

### 3. Color as literacy
Clinical cool whites + desaturated teal stripe for Medical identity; **amber only** at Isolation; red limited to tiny standby / alarm accents — avoid map-wide red bloom.

### 4. Diegetic paperwork as world response
Fitness-for-duty HOLD sheets, IP-3 sleeve language, pharmacy seal notes teach Soft Open / Isolation rules as workplace residue — symptoms, not quest GPS.

### 5. Localized dark + practicals
Path guided by cooler corridor practicals and dim Observation bedside pools; Isolation amber as landmark (not full red emergency wash).

### 6. Barricades as panic archaeology
Tarp / abandoned cart / quarantine closet staging = institutional panic, not combat cover design.

## Canon filter (hard)

- Prefer **contamination, quarantine, abandoned procedure, emergency barricades**.
- Avoid ripping Outlast gore as Project Echo identity.
- Witness remains presence / uncertainty — not chase AI.
- No commercial mesh / layout / trademark reuse.

Rooms dressed as unique workplace cells: Transfer/Security Med Check, Nurse Station, Triage, Exam, OH Console, Records, Pharmacy, Quarantine Closet, Process Corridor, Isolation, Observation, Vent Access — each object justified as clinical or containment residue.
