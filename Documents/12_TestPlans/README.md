# 12_TestPlans

## Purpose

QA test plans and regression checklists for Project Echo features and vertical slice milestones.

## Allowed Assets

- Test plan documents (Markdown preferred)
- Regression suites, pass/fail records as adopted

## Examples

- Future: `TP_Interaction.md`, `TP_VerticalSlice.md`

## Best Practices

- Include Preconditions, Steps, Expected Result, Pass Criteria, Regression Tests.
- Update when a feature ships or breaks.
- Keep prototype smoke tests short and repeatable.
