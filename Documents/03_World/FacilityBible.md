# Facility Bible

Status: Active

Version: 0.1

---

# Purpose

Defines the structure of Asterion Research Institute.

Every room should support both gameplay and storytelling.

---

# Design Philosophy

The institute should feel like a real workplace before it became a horror environment.

Rooms should exist because the facility needed them, not because the game needs them.
