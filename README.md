# PROJECT ECHO

> *"The building remembers."*

Project Echo is a first-person psychological horror game built in **Unreal Engine 5.8**. The game focuses on environmental storytelling, memory-driven gameplay, exploration, and a reactive entity known as **The Witness**.

---

## 🎯 Vision

Project Echo aims to deliver a psychological horror experience where the environment itself becomes the primary storyteller. Rather than relying on constant jump scares, the game emphasizes atmosphere, mystery, and the consequences of the player's actions.

Core Pillars:

- 🧠 Psychological Horror
- 🏢 Environmental Storytelling
- 🔊 Dynamic Audio Design
- 👁️ Reactive AI
- 🧩 Logical Puzzle Design
- 💾 Professional Production Pipeline
